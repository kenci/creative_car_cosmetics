# Creative Car Cosmetics

Premium Landingpage fuer Creative Car Cosmetics (Vite + React + Tailwind CSS).

## Features

- Hero, Leistungen, Galerie (Vorher/Nachher), Bewertungen, Kontakt
- Performante Bildauslieferung und optimierte Assets
- Responsives Layout mit Tailwind CSS und Framer Motion

## Lokale Entwicklung

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Struktur

- `src/components/home`: Sections der Landingpage
- `src/components/common`: Wiederverwendbare Bausteine (Slider, Logo)
- `src/pages`: Seiten und Layout
